## 구름 IDE 사용법

파일 만들기 

```shell
touch 파일이름.c
```



실행파일 만들기

```shell
gcc -o 실행파일이름 파일이름.c

// 예시
// helloworld.c로 실행파일을 만드려면
// gcc -o myprogram helloworld.c
```



생성된 실행파일 실행시키기

```shell
./실행파일이름

// 예시
// 위에서 만든 myprogram을 실행시키려면
// ./myprogram
```
